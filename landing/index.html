What You Get

Protection built for everyday prompting

Prompt Interception

Hooks into submit buttons, form submits, and Enter key sends on major LLM interfaces.

Deep PII Detection

Detects names, ages, emails, phone numbers, SSNs, addresses, DOBs, cards, IDs, IBANs, and API keys.

False Positive Controls

Balanced mode scores findings and reduces noise from weak, isolated signals.

Code-Aware Screening

Ignores weaker signals inside code blocks while still flagging high-risk secrets.

Site Trust List

Whitelist trusted hosts directly from the extension popup for smoother team workflows.

Transparent UX

Clear "Block + Review" or "Warn Only" prompts so users stay in control at send time.

How It Works

Install

Get running locally in under a minute

  1. Open chrome://extensions and enable Developer mode.
  2. Click Load unpacked.
  3. Select /Users/sven/Code/NodeJS/llm-privacy.

FAQ

Quick answers

Does this send my prompt to external servers?

No. Detection is performed locally in the extension runtime.

Can I avoid noisy warnings?

Yes. Use Balanced or Lenient detection and maintain a trusted-site list.

Can I use this on multiple LLM websites?

Yes. The extension supports common LLM interfaces and can be expanded in manifest.json.
